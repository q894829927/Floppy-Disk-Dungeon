# 1.44MB：最后的软盘

一款能够装进 1.44MB 软盘的微型 Windows 原生回合制 Roguelike 游戏。

## 开始游戏

运行 `dist/FloppyDungeon.exe` 即可。在普通的 Windows 10/11 系统上无需安装，也不需要附带额外运行库。

### 操作方式

- 方向键 / `WASD`：移动；撞向敌人时进行攻击
- 按住方向键并按 `F`：安装 `FIREBALL.BIN` 后向对应方向发射火球
- `M`：打开模块管理器
- `H`：查看帮助
- `Esc`：返回、回到标题画面或退出
- `R`：重新开始本局游戏

## 游戏目标

寻找黄色的钥匙芯片，然后前往绿色出口。玩家每行动一次，所有敌人也会行动一次。通过十层随机地牢并消灭最终的系统错误，即可完成游戏。

游戏的可执行程序和全部内容均由代码生成，包括程序化地图、几何图形像素画面以及实时合成音效。

能力模块会占用虚拟 1.44MB 软盘中最后的 160KB 空间。剩余容量不足时，必须删除已经安装的模块，才能安装新的能力。

## 从源码构建

在项目目录中打开 PowerShell，然后运行：

```powershell
.\build.ps1
```

构建脚本会调用本机安装的 Visual Studio C++ 工具链，并使用针对体积优化的编译及链接参数。生成的程序位于：

```text
dist\FloppyDungeon.exe
```

如果可执行文件超过 1,474,560 字节（1.44MB 软盘的二进制容量），构建脚本会直接报错。

## 分发

`FloppyDungeon.exe` 是一个独立的 Windows x86 可执行文件，直接复制或压缩后发送即可。游戏首次运行时会在程序旁边创建一个很小的 `floppy.sav` 存档文件，用于记录最高到达层数和已解锁模块。

程序目前没有数字签名，因此在其他电脑上首次运行时可能出现 Windows SmartScreen 提示。
